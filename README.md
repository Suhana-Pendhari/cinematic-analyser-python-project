# Cinematic Analyzer - Visualize the story behind every rating

A Python desktop application designed to transform IMDb movie datasets into interactive visualizations. Using Tkinter, Pandas, and Matplotlib, it enables users to explore movie trends through year-wise ratings, genre comparisons, and top-rated films via an intuitive graphical interface.
